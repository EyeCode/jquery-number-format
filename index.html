    Jquey Number Format

    Project maintained by EyeCode Hosted on GitHub Pages — Theme by mattgraham

    Welcome to JQuery Number Format Component.

    Easy way to use it:

    $.numberFormat(source[, options]);

    exemple:

    var result = $.numberFormat(5.5555555, {
            prefix: '',
            suffix: ' $',
            digits: 2,
            rounding: true
        })
    // result => 5.56 $
